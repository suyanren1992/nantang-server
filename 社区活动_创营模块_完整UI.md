## 7 步总览

```
① 创营仪式     营队名/期数/类型/主题 → 烛火动画
② 时间参数     日期/日程表/里程碑
③ 财务参数     收入/支出/住宿餐费/预测损益
④ 任务参数     任务池/类别筛选/统计
⑤ 人员选择     从注册用户选待选共建人 → 滚动列表
⑥ 分包确认     每项参数打✓确认 → 全体确认后启营
⑦ 启营仪式     发包+星光汇聚+正式启动
```

## ⑤ 人员选择 — 成为共建人

**管理员视角。** 这一步的核心不是「分配任务」，而是**让一个人成为共建人**——点了「确认认领」，他的身份才变成共建者。认领之前，他可能是冒险者/NPC/云村民。

顶部是「虚位以待」的空圆圈。每确认一个人成为共建人，一个空圆圈就变成头像 + 名字，有仪式感。

**关键交互：**

- 筛选栏：`[全选当前]` + 类别下拉，和 PC 端一致。
- 未认领任务默认折叠。全部分配完毕 → 下一步可用 → 进入⑥分包确认。
