## 4. 全局毛玻璃统一（只做三处）

**原则：字越密越不透。玻璃只用在数字大/信息疏的位置。**

### NT 余额卡片

三个 NT 卡片的背景从纯色渐变改为轻玻璃：

```css
background: rgba(255,255,255,.88);
backdrop-filter: blur(10px);
-webkit-backdrop-filter: blur(10px);
```

数字大（1.2rem），透一点不影响阅读。

### 底部 Sheet

```css
.bottom-sheet{background:rgba(255,255,255,.92);backdrop-filter:blur(16px);-webkit-backdrop-filter:blur(16px);border-radius:20px 20px 0 0}
```

浮层高于所有内容，必须透。

### 子页面内容区底色

```css
.sub-body{background:linear-gradient(180deg,#f5f0e8,#e8ede6)}
```

不用 blur，只加暖色渐变。子页面（社区副本/任务大厅）用暖白底区分于工作台的绿灰底。

**不做玻璃的地方：** 任务卡片、活动卡片、大厅任务卡片——这些字小（`.68rem`），保持纯白 `#fff`，确保可读性。
