## 2-4 · 跨标签同步 + 退出清理

**文件**：`js/app-data.js` + `nantang-mobile.html`

**① 跨标签 storage 监听**（app-data.js 底部，`AppData.init()` 后追加）：
```javascript
window.addEventListener('storage', function(e) {
  if (e.key === 'nt_app_v2_shared' && e.newValue) {
    try {
      var d = JSON.parse(e.newValue);
      if (d) {
        // ponytail: 两层深度合并，同级数组最后写入者胜出
        Object.keys(d).forEach(function(k) {
          if (typeof d[k] === 'object' && !Array.isArray(d[k]) && typeof AppData._data[k] === 'object') {
            Object.assign(AppData._data[k], d[k]);
          } else {
            AppData._data[k] = d[k];
          }
        });
      }
    } catch(ex) {}
    if (typeof refreshUserUI === 'function') refreshUserUI();
  }
});
```

**② 退出清理**（`logout()` 函数中）：
在现有 `localStorage.removeItem('nt_session')` 后加一行：
```javascript
safeStorage.removeItem('nt_remembered_user');
```

**验证**：两个标签页 → A 改 map_locations.config → B 自动看到变化，且 B 自身未保存的字段不被覆盖。退出登录后 `nt_remembered_user` 不存在。

## 2-6 · DIRTINESS_RATES 改为函数（懒读取）

当前代码：
```javascript
var DIRTINESS_RATES = _mlConfig().dirtiness_rates || { bathroom:15, kitchen:10, hallway:8, studio:8, bedroom:5, laundry:5, storage:3, outdoor:2, field:0 };
```

改为函数，回退值从 AppData seed 读取（不硬编码独立副本）：
```javascript
function _dirtinessRates() {
  var cfg = _mlConfig();
  if (cfg.dirtiness_rates && Object.keys(cfg.dirtiness_rates).length) return cfg.dirtiness_rates;
  if (window.AppData && AppData._data.map_locations && AppData._data.map_locations.config && AppData._data.map_locations.config.dirtiness_rates) {
    return AppData._data.map_locations.config.dirtiness_rates;
  }
  return { bathroom:15, kitchen:10, hallway:8, studio:8, bedroom:5, laundry:5, storage:3, outdoor:2, field:0 };
}
```

找到所有 `DIRTINESS_RATES[` 引用处，替换为 `_dirtinessRates()[`。

**验证**：管理员改脏污率配置 → 地图脏污增长速度立即变化（不刷新页面）。
