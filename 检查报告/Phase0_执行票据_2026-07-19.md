### 票 02：`CURRENT_USER` 裸赋值 → `setCurrentUser()`

**文件**：`nantang-mobile.html`  
**行数**：~50 行  
**做什么**：

1. 在全局变量区新增函数：
```javascript
function setCurrentUser(name){
  CURRENT_USER=name;
  if(window.AppData)AppData._currentUser=name;
  if(typeof currentUser!=='undefined')currentUser={name:name,role:(getUsers()[name]||{}).role||'visitor'};
}
```

2. 搜索所有 `CURRENT_USER = ` 裸赋值，逐一替换为 `setCurrentUser(...)`：
   - `enterVillage()` 注册路径：`CURRENT_USER = n` → `setCurrentUser(n)`
   - `enterVillage()` 登录路径：`CURRENT_USER = ln` → `setCurrentUser(ln)`
   - `logout()`：`CURRENT_USER = ''` → `setCurrentUser('')`（需在 setCurrentUser 中处理空字符串）

3. `setCurrentUser('')` 作为登出清理触发：清 `AppData._currentUser`、清 `currentUser`

**验证**：
```
grep -n "= CURRENT_USER\b" nantang-mobile.html → 无匹配
grep -n "setCurrentUser(" nantang-mobile.html → 3处（注册、登录、登出）
```
