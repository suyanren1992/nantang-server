# 登录注册闭环方案

> 2026-07-18 · 让登录/注册走 auth.js 真实验证，NT 系统初始化用户账户

---

## 一、现状

```
现在：点"登录"或"确认注册" → enterVillage() → 直接进村口
      不验密码，不建用户，NT 余额永远是 0
      CURRENT_USER 始终 = '砚仁'

auth.js 已有: registerUser() / loginUser() / getUsers()
      但这些函数从未被调用过 —— 纯摆设
```

## 二、改什么

### 改动 1：`enterVillage()` 分流

```javascript
function enterVillage() {
  var isReg = !document.getElementById('scrRegister').classList.contains('hidden');
  var isLogin = !document.getElementById('scrLogin').classList.contains('hidden');
  
  if (isReg) {
    // 注册流程
    var name = document.getElementById('regName').value.trim();
    var pwd = document.getElementById('regPwd').value.trim();
    if (!name) { alert('请输入名字'); return; }
    if (pwd.length < 6) { alert('密码至少6位'); return; }
    
    // 调用 auth.js 创建用户
    var user = registerUser(name, pwd, 'visitor', 0, '');
    if (!user) { alert('注册失败，名字可能已存在'); return; }
    
    // NT 系统注册账户
    if (window.NT) NT.registerUser(name);
    
    CURRENT_USER = name;
  }
  else if (isLogin) {
    // 登录流程
    var name = document.getElementById('loginName').value.trim();
    var pwd = document.getElementById('loginPwd').value;
    if (!pwd) { alert('请输入密码'); return; }
    
    // 调用 auth.js 验证
    var user = loginUser(name, pwd);
    if (!user) { alert('密码错误或用户不存在'); return; }
    
    CURRENT_USER = name;
  }
  
  // 进入村口
  document.getElementById('loginPage').classList.add('hidden');
  document.getElementById('villagePage').classList.remove('hidden');
  initCarousel();
  setTimeout(initSpcCard, 200);
  refreshUserUI();
}
```

### 改动 2：工作台顶栏显示 NT 余额

```javascript
// 在 refreshUserUI() 或 showMy() 中
var ntVal = document.querySelector('.my-nt-val');
if (ntVal && window.NT) {
  var bal = NT.getUser(CURRENT_USER)?.ntBalance || 0;
  ntVal.innerHTML = '<img src=豆子.png style=width:18px;height:18px;vertical-align:middle;margin-right:2px alt=>' + bal;
}
```

## 三、对原进度的影响

| 已有功能 | 影响 |
|---------|------|
| 任务系统 TASKS | ✅ 无影响（publisher 用 CURRENT_USER 动态） |
| 账本 LEDGER | ✅ 无影响 |
| 物品 MOCK_ITEMS | ✅ 无影响 |
| 地图 Game 桥接 | ✅ `Game.getUser()` 返回 CURRENT_USER |
| NT 系统 | ✅ 用户注册后 NT.earn() 可正常写入 |
| 草稿 DRAFTS | ✅ publisher 用 CURRENT_USER |
| 结算 confirmSettle | ✅ settler=CURRENT_USER 正确 |

**唯一需要注意的**：切换用户时需要重置所有内存状态（`_fromQuestHall`、`_ledgerOpen`、`_itemFold`），否则上一个用户的数据残留在内存。

## 四、验证

1. 打开页面 → ✨我是新来的 → 填名字+密码 → 确认注册 → 进村口
2. 工作台顶栏显示 NT 余额（初始 0）
3. Console 跑 `NT.verify()` → `{pass: true}`
4. 退出 → 重新登录 → 输入密码 → 验证通过才进村
5. 输入错误密码 → 提示"密码错误"
