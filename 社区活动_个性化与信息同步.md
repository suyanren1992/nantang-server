# 社区活动 · 个性化工作台与信息同步
> 🔴 未实现 — 设计完成，代码尚未编写。

> 核心问题：不同共建者领了不同任务包，每天做的事不同。平台如何让每个人看到自己需要的信息？

---

## 一、个性化逻辑

### 任务包 → 工作区块映射

PC 端任务有 7 个类别。每个共建者领的任务包属于哪些类别，决定他概览 Tab 里显示哪些区块。

```
任务类别      对应的工作内容            概览 Tab 显示
─────────────────────────────────────────────────
宣传          拍照/视频/文稿/简报        📸 媒体素材 · 📝 文稿征集
课程          教学/材料/学员进度          📚 课程安排 · 🎨 材料准备
生活          采购/物资/食堂/住宿         📦 采购清单 · 🧾 报销 · 🏨 住宿管理
财务          预算/NT池/结算              📊 NT池 · 🧾 应付未付
管理          审核/发布/签到/协调         ⏳ 待审核 · 📋 发布任务 · ✅ 签到
结项          展览/报告/数据打包          📊 结项进度 · 📸 展览准备
```

## 二、团队动态 — 信息同步

### 动态内容

所有营队成员的操作自动生成一条动态。按类型分类：

| 类型 | 示例 | 图标 |
|---|---|---|
| 任务审核 | 砚仁 通过了「宣传工作」审核 | ✅ |
| 任务提交 | 小云 提交了「厨房打扫」 | 📤 |
| 任务发布 | 小云 发布了「打扫画室」任务 | 📋 |
| 财务结算 | 砚仁 结算了「厨房打扫」5 NT | 💰 |
| 物资采购 | 朝林 采购了宣纸10刀 ¥200 | 📦 |
| 报销提交 | 朝林 提交了颜料报销 ¥150 | 🧾 |
| 媒体上传 | 淑惠 上传了3张活动照片 | 📸 |
| 成员签到 | 朝林 签到入营 | 👋 |
| 问题反馈 | 匿名用户 反馈了「空调不制冷」 | 💬 |
| 日程变更 | 小云 调整了7/21下午安排 | 📅 |

### 动态展示

```
👥 团队动态                    [全部 ▼]

📋 小云 发布了「打扫画室」任务 · 50 NT
   10分钟前

📸 淑惠 上传了3张活动照片
   [缩略图1] [缩略图2] [缩略图3]
   1小时前

💰 砚仁 结算了「厨房打扫」· 5 NT → 小云
   2小时前

📦 朝林 采购了宣纸10刀 · ¥200
   3小时前

```

**筛选：** [全部] [审核] [财务] [物资] [媒体] [签到] [反馈]

**可点：** 点动态里的任务名 → 跳到任务详情。点用户名 → 跳到成员详情。

## 三、底部栏修订

### 最终方案

**底部栏不是通用的——按角色切换。** 切换规则：根据 `currentUser.role` 决定按钮内容。

```
共建者 (builder)：
┌──────────────────────────────────────┐
│ 👤 我的  │  📅 今日  │  📬 消息  │  ＋  │
└──────────────────────────────────────┘

冒险者 (adventurer)：
┌──────────────────────────────────────┐
│ 👤 我的  │  📅 今日  │  🥬 订餐  │  🏨 住宿 │
└──────────────────────────────────────┘

NPC / 云村民（无营地）：
不显示营地底部栏，回退到村口角色卡。
```

| 按钮 | 共建者 | 冒险者 |
|---|---|---|
| 👤 我的 | ✅ 工作台 | ✅ 工作台 |
| 📅 今日 | ✅ 日程Tab | ✅ 日程Tab |
| 📬 消息 | ✅ 待审核/报销通知 | — |
| ＋ | ✅ 角色快捷菜单 | — |
| 🥬 订餐 | — | ✅ 每日要订 |
| 🏨 住宿 | — | ✅ 随时查看 |

**角色升级时底部栏自动切换。** 冒险者成为共建者 → `currentUser.role` 变为 `builder` → 底部栏从冒险者版切换到共建者版。

**营队任务 vs 个人委托（两类体系）：**
- 营队任务：发出者 = 营队名（如「第四期共创营」），类型 = 6 种标签（主线/支线/共创/隐藏/身份/彩蛋）。工作台 `[🏕️营队]` Tab
- 个人委托：发出者 = 个人用户名，标签 = 发布者自定（日常/周期/一次性…）。工作台 `[📋委托]` Tab

## 四、每个角色的完整交互流

### NT管理（小云）

```
概览 Tab：
  待你审核 → [通过]/[退回] → 动态更新
  待你结算 → [结算] → 动态更新
  NT池 → 实时数字

＋ 菜单：
  发布任务 → 发布表单
  成员签到 → 签到列表
  问题反馈 → 反馈列表

📬 消息：
  新的提交审核 → 通知 → 点进去审核
```

### 后勤（朝林）

```
概览 Tab：
  今日采购 → [完成]/[延期]
  待报销 → [提交报销单]

＋ 菜单：
  添加采购项 → 采购表单
  提交报销 → 报销表单（物品/金额/发票照片）
  查看物资库存 → 库存列表
  查看住宿 → 住宿页面

📬 消息：
  报销通过/退回 → 通知
```

### 宣传（淑惠）

```
概览 Tab：
  今日拍摄 → 时间点提醒
  待上传 → [上传]

＋ 菜单：
  上传照片 → 相册选择
  上传视频 → 文件选择
  征集文稿 → 发布征集
  写简报 → 编辑器

📬 消息：
  文稿征集有新的提交 → 通知
```
