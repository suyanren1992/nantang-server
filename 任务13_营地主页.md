## 快捷入口的行为

| 入口 | 可见条件 | 点击行为 |
|---|---|---|
| 📋 我的任务 | isMember | 切换到 `myTabTasks` |
| 📅 日程安排 | isMember | 打开时间线 overlay |
| 👥 成员 | isMember | 切换到社区→成员 Tab |
| 🏆 排行榜 | isMember | 打开排行榜 overlay |
| 💰 资金 | builder/admin | 切换到 `myTabLedger` |
| 🧾 结算 | builder/admin | 打开结算列表 overlay |
| ⚙️ 营地管理 | admin | 打开营地管理 overlay |

（对照 PC 端 `ROLE_CAPABILITIES` 的 `tabs` 数组）
