## 1. 社区公约

### 1.1 签署时机与流程

在地成员入住流程：**选房 → 确认入住 → 签署公约 → 完成入住**

签署弹窗两步式（替代现在的"一句话确认"）：

1. **第一步**：展示公约全文（可滚动阅读，分章节）
2. **第二步**：勾选"我已阅读并同意"→ 确认
3. 签署后信箱收到通知：`📜 你已签署南塘社区公约 · {日期}`

### 1.2 公约正文（初稿，可公约会议修订）

1. **厨房使用规则**：使用后及时清洁归位；公共食材标注放入日期；过期物品人人可清理
2. **卫生打扫规则**：各区域按轮值表打扫；看到脏了随手清理是被鼓励的美德
3. **日常生活规范**：尊重他人作息与隐私；公共空间保持可用状态
4. **安全责任声明**：⚠️ 社区不承担人身安全保障——猫狗抓伤等意外由自己负责；付了住宿费 ≠ 买了保险；请自己照顾好自己
5. **大扫除义务**：住满 7 天必须参加每周一次的大扫除
6. **退出机制**：（待公约会议补充）

### 1.3 技术实现

- 公约正文以结构化文本（分章节）存放在 `js/data.js`，一处维护多处引用
- 全貌页的 📜 社区公约入口（`_openCovenantOverlay()`）从"只显示定价标准"扩展为"公约全文 + 定价标准附录"

## 2. 新手引导 · 层面一：生活融入任务包

**目的**：帮新入住成员挣到第一笔 NT，同时完成对社区生活的初步融入。

### 2.1 任务包内容（签署公约后默认发放）

| # | 任务 | 建议奖励 |
|---|---|---|
| 1 | 在社区各处打卡（地图标记点看一看） | +5 NT |
| 2 | 参加一次帮厨 | +10 NT |
| 3 | 参加一次晨练 | +5 NT |
| 4 | 发起或参加一次饭局 | +10 NT |
| 5 | 陪老人说说话 | +5 NT |
| 6 | 关爱一下小朋友 | +5 NT |
| 7 | 参加一次乡亲大会（每月初一、十五各一场，有老人来） | +5 NT |

全部完成：额外称号 + 奖励（延续 v11 新手任务设计）。

### 2.2 任务流转规则

```
默认发放 ──▶ 成员接受 → 进入「任务台」跟踪
        └─▶ 不接受 → 躺在「信箱」里，随时可取出完成
                    ├─ 嫌碍眼 → 可删除
                    └─ 删除后想找回 → 找管理员重新申请
管理员：有权向任何成员发布新手任务
```

## 3. 新手引导 · 层面二：平台功能引导 NPC

**目的**：降低每个功能的使用门槛——卡片室、校核、大扫除、社区副本、地图，每个功能都是一间"room"，NPC 负责把门推开。

### 3.1 形态

- 一个**管理员头像的小 NPC**（用砚仁的头像），带对话框
- 出现在各页面关键位置，像游戏引导员一样讲解："这是干嘛的"、"点这里可以……"
- 对话可关闭、可跳过；看过的引导不再重复（可重置）

### 3.2 覆盖场景（按优先级）

1. 注册后首次进村：整体导览（村口三卡是什么）
2. 工作台：任务/账本/物品三个 Tab
3. 卡片室：怎么发起发现、怎么猜、NT 从哪来
4. 实景地图：房间状态灯、打扫、物品
5. 社区副本：营地、活动、档案室

### 3.3 实现要点

- 引导脚本以 JSON 数据驱动（页面 → 锚点 → 文案），不写死在组件里
- 与现有 `#newbieGuideOverlay` 新手浮层合并演进，注意 z-index 层级（参考 B3 教训）
