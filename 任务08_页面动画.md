# 任务 08 · 页面切换动画

> **目标：** 页面切换有过渡动画 + 村口卡片点击有激活反馈
> **改什么：** CSS + `showMy()` / `showVillage()` 函数

---

## 1. 页面切换动画

当前 `showMy()` 和 `closeMyPage()` / `showVillage()` 是瞬间切。overlay 已有 `fadeIn` 动画，但 myPage 进出没有。

### 增加 CSS

```css
.my-page{transition:transform .3s cubic-bezier(.4,0,.2,1),opacity .3s}
.my-page.slide-in{transform:translateY(0);opacity:1}
.my-page:not(.slide-in){transform:translateY(30px);opacity:0}
```

### 修改 `showMy()`

```javascript
function showMy(){
  document.getElementById('villagePage').classList.add('hidden');
  var p = document.getElementById('myPage');
  p.classList.remove('hidden');
  p.classList.remove('slide-in');
  requestAnimationFrame(function(){ p.classList.add('slide-in'); });
}
```

### 修改 `closeMyPage()` — 加延迟隐藏

```javascript
function closeMyPage(){
  var p = document.getElementById('myPage');
  p.classList.remove('slide-in');
  setTimeout(function(){
    p.classList.add('hidden');
    if(_fromQuestHall){_fromQuestHall=false;document.getElementById('overlayQuestHall').classList.add('open')}
    else{document.getElementById('villagePage').classList.remove('hidden')}
  }, 250);
}
```

## 2. 村口卡片激活反馈

点「进入」按钮时，卡片先闪烁再跳转。

### CSS

```css
.page-card.activating{transform:scale(1.04);border-color:rgba(160,200,170,.8);box-shadow:0 0 0 4px rgba(61,107,82,.35),0 12px 48px rgba(0,0,0,.25);transition:transform .15s,box-shadow .15s,border-color .15s}
```

### 修改三个按钮的 onclick

不是改函数，而是在按钮的 onclick 中先触发激活，再延迟调函数。例如：

```html
<button class="vp-btn" onclick="var c=this.closest('.page-card');c.classList.add('activating');setTimeout(function(){c.classList.remove('activating');openCommunityPage();},150)">📋 进入副本</button>
```

## 验证

- 村口 → 我的：页面从下方轻推滑入（300ms）
- 我的 → 村口：页面淡出后村口出现
- 村口卡片点「进入」：卡片先放大 + 绿光（150ms），再跳转
