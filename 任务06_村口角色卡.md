# 任务 06 · 村口角色卡

> **目标：** 村口 footer 从三按钮换成毛玻璃角色卡
> **参考：** `项目/南塘云村App/原型_导航骨架演示.html` 第 79-86 行 + 第 245-249 行

---

## 当前

```html
<div class="village-footer">
  <button onclick="showMy()">👤 我的</button>
  <button>⚙️ 设置</button>
  <div class="vf-user">demo · 预览</div>
</div>
```

## 改为

**毛玻璃卡片内只有头像 + 名字 + 角标。** 身份/勋章/天数放在卡片下方一行半透明文字。

```
┌──────────────────┐
│ 👤 砚仁     [3]  │  ← 毛玻璃卡片：头像 + 名字 + 红色角标
└──────────────────┘
  🧱 共建者 · 🏅 · 120天   ← 卡片下方，半透明
      ⋯                    ← 引导提示（首次显示）
```

**交互层次：**
- 单击 → 进入工作台
- 长按 600ms → 弹出菜单（👤进入工作台 / ⚙️设置 / 🚪退出登录）

**如何让人知道可以点：**
1. 卡片有悬浮阴影 + 微浮动动画（`animation: cardFloat 3s ease-in-out infinite`），暗示可交互
2. 首次打开村口时，卡片下方显示「⋯」三点的呼吸动画，3 秒后自动消失——暗示「这里还有更多」
3. 红色角标本身是视觉锚点——人的视线会被红点吸引，自然注意到卡片

### 4. 长按菜单 JS

在内联 `<script>` 中加：

```javascript
var _spcPressTimer = null;
document.addEventListener('DOMContentLoaded', function() {
  var card = document.querySelector('.square-player-card');
  if (!card) return;
  card.addEventListener('mousedown', function(){ _spcPressTimer = setTimeout(showPlayerMenu, 600); });
  card.addEventListener('mouseup', function(){ clearTimeout(_spcPressTimer); });
  card.addEventListener('mouseleave', function(){ clearTimeout(_spcPressTimer); });
  card.addEventListener('touchstart', function(e){ _spcPressTimer = setTimeout(showPlayerMenu, 600); });
  card.addEventListener('touchend', function(){ clearTimeout(_spcPressTimer); });
  card.addEventListener('touchmove', function(){ clearTimeout(_spcPressTimer); });
  card.addEventListener('click', function(e){
    if (_spcPressTimer) { clearTimeout(_spcPressTimer); showMy(); }
  });
});

function showPlayerMenu() {
  var sheet = document.getElementById('bottomSheet');
  var body = document.getElementById('sheetBody');
  var title = document.getElementById('sheetTitle');
  title.textContent = '';
  body.innerHTML = '<div style="padding:8px 0">'+
    '<div style="padding:12px 16px;font-size:.9rem;cursor:pointer;border-bottom:1px solid #e5ddce" onclick="NM.closeSheet();showMy()">👤 进入工作台</div>'+
    '<div style="padding:12px 16px;font-size:.9rem;cursor:pointer" onclick="NM.closeSheet();NM.showConfirm(\'退出登录\',\'确定要退出吗？\',function(){logoutUser();location.reload()})">🚪 退出登录</div>'+
    '</div>';
  sheet.classList.remove('hidden');
  document.getElementById('sheetBackdrop').classList.remove('hidden');
}
```

> **注意：** 如果 `NM.closeSheet` 等 NM 函数不存在，复用已有的 `closeSheet()` 或直接操作 DOM。`logoutUser()` 来自 `auth.js`。
