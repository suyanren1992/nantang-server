# 任务 09 · 头像 + 顶栏信息优化

> **目标：** 头像边框按角色换色 + 顶栏文字重新排版。不做等级环。
> **改什么：** CSS + 工作台顶栏 HTML（第 189-196 行）

---

## 当前

头像 34px + 绿色边框 2px。顶栏文字全挤在一行：

```
🌐 | 👤 砚仁 管理员 120天 🏅 🫘1250
      🏕️ 第四期共创营
```

DiceBear 头像太小（34px），脸看不清。角色混在名字后面容易忽略。

---

## 改为

头像放大到 40px，边框 2.5px 按角色变色。文字分行排版：

```
←  ┌──────────┐  砚仁 🏅
    │ DiceBear │  🧱 共建者 · 来了120天
    │   头像   │  🫘 1,250 NT
    └──────────┘
                   🏕️ 第四期共创营
```

---

## 改什么

### 1. CSS 改动

```css
/* 头像放大 + 角色色边框 */
.my-avatar{width:40px;height:40px;border-radius:50%;overflow:hidden;flex-shrink:0;border:2.5px solid #3d6b52;background:#c8d8c8}

/* 角色色变体 — 用 JS 动态加 class 或直接改 border-color */
.my-avatar.role-admin{ border-color:#c88740 }     /* 金色 */
.my-avatar.role-builder{ border-color:#5d8c52 }   /* 绿色 */
.my-avatar.role-adventurer{ border-color:#4a7a82 } /* 蓝色 */
.my-avatar.role-npc,.my-avatar.role-visitor{ border-color:#8a8a8a } /* 灰色 */

.my-avatar img{width:100%;height:100%;object-fit:cover}

/* 顶栏信息排版 */
.my-id-area{flex:1;min-width:0;line-height:1.4}
.my-id-row1{display:flex;align-items:center;gap:6px}
.my-name{font-weight:700;font-size:.85rem}
.my-role{font-size:.72rem;color:#5a6e5c}
.my-nt-val{font-weight:700;color:#3d6b52;font-size:.85rem;white-space:nowrap;margin-left:auto}
.my-id-row2{font-size:.7rem;color:#5a6e5c}
.my-id-row3{font-size:.65rem;color:#5a6e5c}
```

### 2. HTML 替换（当前 my-topbar 区域）

当前顶栏是（约 296-299 行）：

```html
<button onclick="closeMyPage()" style="...">✕</button>
<div class="my-avatar" style="width:34px;height:34px"><img src="...dicebear..."...></div>
<div style="flex:1..."><span>砚仁</span><span>管理员</span><span>120天</span>...<span>🏕️ 第四期共创营</span></div>
<span>🏅</span>
<div>🫘1,250</div>
```

替换为三行排版：

```html
<button onclick="closeMyPage()" style="background:none;border:none;cursor:pointer;font-size:1.2rem;color:#3d6b52;padding:4px 8px;line-height:1">✕</button>
<div class="my-avatar role-admin" id="hudAvatar" style="width:40px;height:40px">
  <img src="https://api.dicebear.com/7.x/avataaars/svg?seed=砚仁&size=40" width="40" height="40" style="border-radius:50%;object-fit:cover" alt="">
</div>
<div class="my-id-area">
  <div class="my-id-row1">
    <span class="my-name">砚仁</span>
    <span class="my-medals">🏅</span>
    <span class="my-nt-val">🫘 1,250</span>
  </div>
  <div class="my-id-row2">
    <span class="my-role-tag">🧱 共建者</span>
    <span class="my-days">· 来了 120 天</span>
  </div>
  <div class="my-id-row3">🏕️ 第四期共创营</div>
</div>
```

**变化：** 头像 34→40px，边框 2→2.5px + 角色色。✕ 按钮保留。三行文字层次清晰。NT 移到右端。
