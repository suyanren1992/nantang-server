# 任务 01 · 接入 DiceBear 头像库

> **目标：** 所有头像从静态图片换成 DiceBear API 动态生成
> **改什么：** `nantang-mobile.html` 内联 `<script>` 块
> **不改：** HTML 结构、CSS

---

## 当前状况

所有头像用的是同一张静态图片 `92e279b22d04d4c831aec15aa09213df (1).png`。登录页、工作台顶栏、用户芯片——全部同一张图。

---

## 要改成什么

用 DiceBear API 根据用户名生成不同头像。调用方式：

```
https://api.dicebear.com/7.x/avataaars/svg?seed=用户名&size=80
```

同一个 seed（用户名）永远生成同一张脸。不同用户名 = 不同脸。

---

## 改什么

### 1. 新增头像生成函数

在内联 `<script>` 顶部（约第 256 行，`var LEDGER` 之前）加：

```javascript
var AVATAR_STYLE = 'avataaars';
function avatarURL(seed, size) {
  return 'https://api.dicebear.com/7.x/' + AVATAR_STYLE + '/svg?seed=' + encodeURIComponent(seed) + '&size=' + (size||80);
}
function avatarImg(seed, size) {
  return '<img src="' + avatarURL(seed, size) + '" width="' + (size||40) + '" height="' + (size||40) + '" style="border-radius:50%;object-fit:cover" alt="" onerror="this.style.opacity=\'0\'">';
}
```

### 2. 替换所有头像引用

以下位置把 `<img src="92e279...png">` 换成 `avatarImg('用户名', 尺寸)`：

| 位置 | 当前 | 改为 |
|---|---|---|
| 第 127 行 登录页 Logo | `<img src="...png">` | 保持不变（这是 Logo，不是头像） |
| 第 159 行 用户列表 demo 芯片 | `<img src="...png">` | `avatarImg('demo', 48)` |
| 第 170 行 村口品牌区 Logo | `<img src="...png">` | 保持不变（Logo） |
| 第 192 行 工作台顶栏头像 | `<img src="...png">` | `avatarImg('砚仁', 34)` |

**注意：** 以后接入真实数据时，`'砚仁'` 换成 `currentUser.name`。现在先写死确保 UI 能跑。

### 3. DiceBear API 说明

无需注册、无需 API key、无请求次数限制。svg 格式体积小（约 2KB/张），移动端友好。

---

## 验证

1. 打开 `nantang-mobile.html`
2. 登录页 → 用户芯片的 demo 头像应显示 DiceBear 生成的脸（不是原来那张静态图）
3. 进村口 → 工作台顶栏头像应显示不同的脸（seed 不同则脸不同）
4. 断网时头像消失但不报错（`onerror` 兜底）
