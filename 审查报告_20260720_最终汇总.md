## 二、所有问题的共同根因

**核心根因：`app.js`（2081 行地图模块）是一个完全独立运行的硬编码演示系统。** 它有自己的数据、自己的配置、自己的存储，与 AppData、API、NT 经济系统之间**没有数据连接**。

```
app.js（独立世界）           你建的 AppData/API 系统
─────────────────────        ──────────────────────
HARDCODED_BUILDINGS          AppData._data.map_locations
_defaultConfig() 60+ 数值    server/routes/* 43 个端点
MGMT_DATA → localStorage     NT 经济 / 会计等式
从不调用 refreshUserUI()     refreshUserUI 能更新一切
从不读取 API                 服务端已有完整数据
```

你在地图上看到的所有内容——在线人数、整洁状态、入住信息、代收费用——**全部是写死在代码里的字符串，永远不变。**
