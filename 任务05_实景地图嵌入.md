---

## 改什么

### 1. 复制 + 引入地图文件

```
复制 实景地图移动端代码/app.js → js/map-app.js
复制 实景地图移动端代码/theme.css → css/map-theme.css
```

在 `<script src="js/seed-test-data.js">` 之后加：

```html
<link rel="stylesheet" href="css/map-theme.css">
<script src="js/map-app.js"></script>
```

如 `map-theme.css` 和 `variables.css` 的 `:root{}` 冲突 → 删 `map-theme.css` 开头的变量块。

### 2. 替换 overlay-body 占位内容

找到 `overlayMap` 的 `overlay-body` div（约第 427 行），把占位文字替换为容器 div：

```html
<div class="overlay-body"><div id="mapEmbedContainer" style="width:100%;height:100%"></div></div>
```

### 3. 修改 `openMapPage()` 函数

地图的 `app.js` 依赖 `window.Game`。在内联 `<script>` 中加 Game 对象（与窗口B mock 接口一致）：

```javascript
window.Game = {
  openTask: function(id) { openTaskDetail(id); },
  toast: function(msg) { /* 后续实现 */ },
  confirm: function(t, m, fn) { if (confirm(t+'\n'+m)) fn(); },
  getUser: function() { return { name: '砚仁', role: 'admin', avatar_seed: 0 }; },
  getData: function() { return window.data || {}; },

```

### 4. 替换 `openMapPage()` 

```javascript
function openMapPage() {
  if (typeof Map !== 'undefined' && Map.init) {
    // 用 subPage 做容器
    document.getElementById('subTitle').textContent = '实景地图';
    document.getElementById('subBody').innerHTML = '<div id="mapEmbedContainer"></div>';
    document.getElementById('subPage').classList.add('open');
    setTimeout(function() {
      Map.init(document.getElementById('mapEmbedContainer'));
    }, 100);
  } else {
    document.getElementById('subTitle').textContent = '实景地图';
    document.getElementById('subBody').innerHTML = '<div style="text-align:center;padding:60px;color:#5a6e5c">🗺️<br><br>地图模块未加载</div>';
    document.getElementById('subPage').classList.add('open');
  }
}
```
