## 五级身份体系

```
🧙 管理员 (admin)       isMember:true   最高权限，不可降级
🧱 共建者 (builder)     isMember:true   需先有冒险者经历
⚔️ 冒险者 (adventurer)  isMember:true   注册时需要邀请码
👥 在地伙伴 (npc)       isMember:false  管理员发NPC邀请码
🏕️ 云村民 (visitor)     isMember:false  直接注册，无邀请码
```

`isMember` 来自 `ROLE_CAPABILITIES`（nt.js 第 22-33 行）。

## 进入营地完整逻辑

```
点击「进入营地」
  │
  ├── 未登录
  │     → 「请先登录」
  │
  ├── 是 admin / builder / adventurer（isMember = true）
  │     → ✅ 直接进入营地主页
  │
  ├── 是 npc（在地伙伴）
  │     → 弹窗：「进入营地需要冒险者身份。」
  │       你有邀请码吗？
  │       ┌──────────┐ ┌──────────┐
  │       │ 输入邀请码 │ │   取消    │
  │       └──────────┘ └──────────┘
  │       输入邀请码 → 验证 → 角色升级为 adventurer → 进入营地
  │
  ├── 是 visitor（游客）
  │     → 弹窗：「进入营地需要冒险者身份。」
  │       你有邀请码吗？
  │       同上逻辑
  │
  └── 无进行中的营队
        → 「暂无进行中的营队。请等待管理员创建。」
```

### 错误提示

| 情况 | 弹窗内容 |
|---|---|
| 邀请码无效 | 「邀请码无效。请检查后重试。」 |
| 邀请码已被使用 | 「此邀请码已被使用。」 |
| 输入格式不对 | 「邀请码格式：NT-XXXXXXXX」 |

### JS 逻辑

```javascript
function enterCamp(campId) {
  // 1. 未登录
  if (!window.currentUser) {
    alert('请先登录');
    return;
  }
  
  // 2. 已是成员 → 直接进
  if (typeof userCan === 'function' && userCan(window.currentUser, 'isMember')) {
    openCampHome(campId);
    return;
  }
  
  // 3. npc 或 visitor → 需要邀请码
  showInviteCodePrompt(function(code) {
    // 验证邀请码
    var wc = typeof getWorldCodes === 'function' ? getWorldCodes() : [];
    if (wc.indexOf(code) === -1) {
      alert('邀请码无效。请检查后重试。');
      return;
    }
    
    // 消耗邀请码 + 升级角色
    var result = typeof changeUserRole === 'function' 
      ? changeUserRole(window.currentUser.name, 'adventurer', { inviteCode: code })
      : { ok: false, error: '功能未加载' };
    
    if (result && result.error) {
      alert(result.error);
      return;
    }
    
    // 成功 → 进营地
    alert('身份升级成功！欢迎加入营地 ⚔️');
    openCampHome(campId);
  });
}

function showInviteCodePrompt(onSubmit) {
  var code = prompt('输入冒险者邀请码（格式：NT-XXXXXXXX）：');
  if (!code) return;
  onSubmit(code.trim().toUpperCase());
}
```

### 邀请码格式

PC 端 `generateInviteCode()` 输出格式：`NT-` + 8 位 36 进制字符，如 `NT-A3F8K2M9`。
