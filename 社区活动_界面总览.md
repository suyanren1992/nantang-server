| [查看回顾] | 任何人 | → 界面3（往期模式） |
| [🌍 世界终端] | 管理员 | → 创营向导第①步 |

## 界面 2：营地主页（进入营地后）

**顶部 Tab 栏切换内容，底部固定操作栏。** 导航与 `共建人营地界面.md` 一致。

```
┌──────────────────────────────────────┐
│ ← 返回    第四期共创营                 │
├──────────────────────────────────────┤
│ [概览] [日程] [资金] [成员] [排行] [⚙️]│  ← 顶部 Tab 栏
├──────────────────────────────────────┤
│                                      │
│         （当前 Tab 内容）              │
│   概览 = 营队数据 + 个性化区块         │
│   日程 = 日期滚轮 + 完整时间线         │
│   资金 = RMB/NT 双金库                │
│   成员 = 列表 + 详情                  │
│   排行 = 贡献排名                     │
│   ⚙️  = 营地管理（仅管理员/NT持有者）  │
│                                      │
├──────────────────────────────────────┤
│ 👤 我的 │ 📅 今日 │ 📬 消息 │ ＋     │  ← 底部栏（共建者）
│ 👤 我的 │ 📅 今日 │ 🥬 订餐 │ 🏨 住宿 │  ← 底部栏（冒险者）
└──────────────────────────────────────┘
```

**操作与结果：**

| 点击 | 结果 |
|---|---|
| Tab 栏切换 | → 切换内容区 |
| 👤 我的 | → 工作台（[🏕️营队] / [📋委托]） |
| 📅 今日 | → 跳到日程 Tab，定位到今天 |
| 📬 消息 | → 消息聚合页 |
| ＋ / 🥬🏨 | → 角色快捷菜单 / 订餐 / 住宿 |

## 完整操作流程

```
村口 → 社区主页 → [进入营地] → 营地主页
                      │               │
                      │               ├── 📅日程 → 日期滚轮/完整时间线
                      │               ├── 💰资金 → 双金库
                      │               ├── 👥成员 → 成员详情
                      │               ├── 🏆排行 → 排名
                      │               ├── ⚙️管理 → 审批/发布/审核/结算/NT池/完结
                      │               └── 点头像 → 工作台 [🏕️营队] [📋委托]
                      │
                      └── [查看窗口] → 营地对外展示 → [我要报名]
```
