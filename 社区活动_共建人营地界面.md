## 入口

工作台顶栏下方新增一个营队横幅。有活跃营队时显示，没有时不显示。

```
┌──────────────────────────────────────┐
│ ←  ✕                                │
│ 👤 小云 🏅  🧱共建者  🫘125 NT      │
├──────────────────────────────────────┤
│ ┌──────────────────────────────┐   │
│ │ 🏕️ 第四期共创营  ·  进行中    │   │  ← 营队横幅，点击进入营地
│ │ ⚠️ 有待确认事项               │   │     有待确认时显示警告
│ └──────────────────────────────┘   │
│                                      │
│ （工作台原有内容）                    │
└──────────────────────────────────────┘
```

点击横幅 → 进入营地详情 overlay。

## 日程设计详述

### 日常查看：日期滚轮 + 当日卡片

底部是横向日期滚轮，顶部是当前日期的安排卡片。左右滑动滚轮切换日期。

```
┌──────────────────────────────────────┐
│ 📅 日程                              │
├──────────────────────────────────────┤
│                                      │
│ ┌──────────────────────────────┐   │
│ │      7月21日                  │   │
│ │                              │   │
│ │   上午 06:00   晨练打卡        │   │
│ │   上午 10:00   工笔画临摹      │   │
│ │   下午 14:00   陶艺课程        │   │
│ │   晚上 19:00   分享晚会        │   │
│ └──────────────────────────────┘   │
│                                      │
│ [📅 完整时间线]                       │  ← 切换按钮
│                                      │
│ ┌──┬──┬──┬──┬──┬──┬──┬──┐         │
│ │  │20│21│22│23│24│25│26│  │  ← 滚轮 │
│ │  │开│● │  │  │  │展│  │  │         │
│ └──┴──┴──┴──┴──┴──┴──┴──┘         │
│        ↑ 今天高亮                      │
│   ← 左右滑动选择日期 →                │
└──────────────────────────────────────┘
```

**滚轮规格：**
- 每个日期一个胶囊：日期数字 + 里程碑标记（开/展/结）
- 今天高亮（墨绿底白字）
- 有里程碑的日期显示小圆点标记
- 默认滚动到今天位置
- 滑动时上方卡片实时切换
