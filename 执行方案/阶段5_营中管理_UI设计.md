### 1.4 区块 3：待审核（提交审核）

共建者完成任务后提交，管理员审核通过/退回。

```
⏳ 待审核（1项）

┌──────────────────────────────────────┐
```

**通过：** 任务状态 → `approved`，进入区块 4「应付未付」。写 `logActivity`。发信箱通知给提交者。
**退回：** 必填原因（input 不为空时才能点退回）。任务状态回退为 `active`。

### 1.5 区块 4：应付未付

```
📋 应付未付（2笔，共 NT 25）

┌──────────────────────────────────────┐
│ 小云 · 厨房打扫 · NT 5               │
```

**逐笔结算：** 点 `[🔓 结算]` → 确认弹窗 → `recordTransaction({type:'task_reward', from: 'camp_pool', to: person, amount: nt})`。任务状态 → `已结算`。NT 池更新。写 `logActivity`。
**一键结算：** 全部应付项批量结算，一次确认。每条独立写 transaction。


### 2.3 映射规则

```javascript
var CATEGORY_BLOCKS = {
  '管理': ['review_queue', 'publish_task'],
  '财务': ['settlement_queue', 'nt_pool'],
  '生活': ['purchase_list', 'reimbursement'],
  '课程': ['class_schedule', 'material_prep'],
  '宣传': ['photo_shoot', 'essay_collect'],
  '结项': ['closing_progress', 'exhibition_prep']
};

function getMyBlocks(userTasks) {
  var categories = userTasks.map(function(t) { return t.category; });
  categories = [...new Set(categories)];  // 去重
  var blocks = [];
  categories.forEach(function(cat) {
    var b = CATEGORY_BLOCKS[cat] || [];
    blocks = blocks.concat(b);
  });
  return blocks.slice(0, 3);  // 最多 3 个，其余折叠
}
```
