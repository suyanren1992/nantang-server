### 1.3 交互

- 点头像 → 选中（绿色边框），下方详情卡切换到该成员
- 「NT 计算方式」→ 点击展开/折叠。默认折叠（大部分人只关心总数）
- 「任务明细」→ 点击展开/折叠。列出所有任务及状态（✅完成/◷进行中/☐未开始）
- 「工资分配」→ 点击展开/折叠。显示 RMB + NT 分配明细

### 1.4 数据来源

- 头像列表 + NT 余额：`NT.getUser(name).ntBalance`（营队 scope）
- 任务明细：`data.tasks` 中该成员 `claimants` 的任务
- 完成率：已完成任务数 / 总任务数
- NT 计算：`任务激励总和 + 角色NT + 活动NT`。角色NT = 按角色类型的基础补贴。活动NT = 参与活动获得的额外奖励
- 工资分配：RMB 从 `c.budget`（`getCampData().budget`）预算中读取（审计 P1：营队启动后 `_campDraft` 已清空）

## 三、结营仪式

### 3.1 触发

管理员点「完结本期」并确认 → 营队状态变 `archived` → 自动弹出结营仪式动画。

### 3.2 敬字亭动画（GSAP Timeline）

```
阶段 1（0.8s）：背景变暗，🔥 敬字亭从底部升入画面中心
               （ease: Power2.easeOut）

阶段 2（1.0s）：火焰在亭中燃起，暖色光晕扩散
               （多层火焰：底层大焰 + 顶层小焰，晃动频率不同）

阶段 3（0.8s）：文字「以字入火，以志成林」逐字浮现
               （白色，stagger 0.08s，从火焰上方向上升起）

阶段 4（1.0s）：火焰渐弱，化为金色星点向四周飞散
               （粒子效果：20 个 4px 金色圆点，随机方向 ease out）

阶段 5（0.6s）：星点汇聚为「数据报告已生成」文字
               → 停留 1s → 自动进入结营报告页
```

总时长约 5 秒。动画期间不可交互。按「跳过」→ 直接进入报告页。

**敬字亭背景：** 这是实景仪式的数字映射。现实中大家在敬字亭烧掉练习稿和笔记。App 里的「完结本期」按钮是仪式的数字触发器。

### 3.3 实现

```javascript
// GSAP Timeline
var tl = gsap.timeline();
tl.from('#jzt', { y: 200, opacity: 0, duration: 0.8, ease: 'power2.out' })
  .to('#jzt-fire', { opacity: 1, scale: 1.2, duration: 1.0 })
  .from('#jzt-text span', { y: -20, opacity: 0, stagger: 0.08, duration: 0.8 }, '-=0.5')
  .to('#jzt-fire', { opacity: 0, duration: 1.0 })
  .to('#jzt-particles', { opacity: 1, duration: 0.6 }, '-=0.5')
  .from('#jzt-report', { opacity: 0, y: 10, duration: 0.6 });
```
