### 1.2 交互细节

**日期切换：** 顶部 `← 7/19 | 📅 7/20（今天）| 7/21 →`。今天高亮。只能切换到有菜单数据的日期（无菜单 = 箭头灰掉）。

**预定：** 点 `[🟢 预定]` → 检查 NT 余额 → 不够 → toast「NT余额不足（需 10 NT）」。足够 → 确认弹窗「预定 7/20 午餐？· 10 NT」→ 确认 → 按钮变为 `✅ 已预定` + `❌ 取消`。写 `recordTransaction({type:'canteen_preorder', amount:10})` + `logActivity('canteen', ...)`。

**取消：** 点 `❌ 取消` → 确认弹窗「取消预定？将退回 10 NT」→ 确认 → 按钮恢复为 `🟢 预定`。写退款 `recordTransaction({type:'canteen_refund', amount:10})`。

**防重复：** 后端检查 `_mealOrders[user_date_meal]`，已预定再次点击 → toast「已预定过了」。

**菜品图标映射：**
```javascript
function getFoodIcon(name) {
  var map = { '饭':'🍚','米':'🍚','面':'🍜','汤':'🍲','豆腐':'🫘','豆':'🫘',
    '菜':'🥬','蔬':'🥬','瓜':'🥒','薯':'🍠','蛋':'🥚','菇':'🍄','饺子':'🥟',
    '沙拉':'🥗','果':'🍎','鱼':'🐟','肉':'🍖' };
  for (var k in map) { if (name.indexOf(k) !== -1) return map[k]; }
  return '🍽️';
}
```

**管理员设菜单：** `setCanteenMenu('2026-07-20', ['糙米饭','清炒时蔬','菌菇汤'], ['手工面','红烧豆腐','紫菜汤'])`。写入 `AppData._data.canteenMenu[date] = { lunch: [...], dinner: [...] }`。

### 1.4 预定时间限制

- 午餐预定截止：当天 10:00。超过 → 按钮灰掉「已截止」
- 晚餐预定截止：当天 16:00。超过 → 同上
- 管理员可随时设菜单，不受限制

### 3.2 数据来源

- 流水：`AppData._data.finance` 中 `scope === 'camp'` 的记录
- 按人汇总：遍历流水，按 `from` / `to` 分组聚合
- 筛选 chip：按 `type` 字段过滤

### 3.3 流水行格式

| 类型 | 图标 | 显示 |
|------|------|------|

| `material` | 📦 | `物资 · {person} · {note} · ¥{amount}` |
| `transport` | 🚗 | `交通 · {person} · {note} · ¥{amount}` |
| `task_reward` | 🎁 | `奖励 · {person} · {taskName} · {amount} NT` |
| `tip` | 💬 | `打赏 · {from}→{to} · {amount} NT` |
| `canteen_preorder` | 🥬 | `订餐 · {person} · {date} {meal} · {amount} NT` |
| 其他 | 📝 | `{note} · {amount}` |
