## 一、营地 overlay 整体布局

```
┌──────────────────────────────────────┐
│ ← 返回    第四期共创营                 │  ← header
├──────────────────────────────────────┤
│ [概览] [日程] [资金] [成员] [排行] [⚙️]│  ← Tab 栏，横向滑动 capsule
├──────────────────────────────────────┤
```

### 1.1 Tab 栏

- 横向滑动 capsule 行。选中态 bg=#3d6b52 white，默认态 transparent #5a6e5c
- 6 个 Tab。管理员/共建者全显示。冒险者 5 个（无 ⚙️管理）

### 1.2 底部栏

| 按钮位 | 共建者 | 冒险者 |
|--------|--------|--------|
| 1 | [DiceBear头像] → `showMy()` | [DiceBear头像] → `showMy()` |
| 2 | 📅 今日 → 跳日程 Tab | 📅 今日 → 跳日程 Tab |
| 3 | 📬 消息 → 信箱面板 | 🥬 订餐 → toast（阶段 4） |
| 4 | ＋ → toast（阶段 5） | 🏨 住宿 → toast（阶段 4） |

第 1 位用当前用户 DiceBear 头像（28px 圆形），不是 emoji。点击 → `showMy()` 打开工作台。工作台顶部栏点头像 → `toggleProfile()` 展开资料卡。**这条链路代码已存在，不新建。**

## 二、Tab 1：概览（导览面板）

不展示内容细节——每个区块是跳转入口。

```
┌──────────────────────────────────────┐
│                                      │
│ ┌──────────────────────────────┐   │
│ │ 📊 营队概览                   │   │  ← 纯信息，不可点
│ │ 7/20-7/27 · 12人 · 8个任务   │   │
│ │ ████████░░░░ 64%             │   │
│ └──────────────────────────────┘   │
│                                      │
│ ┌────────────────────────────┐      │
│ │ 📋 我的任务   3项·33 NT  → │      │  ← 点→工作台+选中[营队]chip
│ └────────────────────────────┘      │
│ ┌────────────────────────────┐      │
│ │ 🕐 今日   06:00 晨练…   → │      │  ← 点→日程Tab+定位今天
│ └────────────────────────────┘      │
│ ┌────────────────────────────┐      │
│ │ 💰 资金   ¥4,500·NT7,320 →│      │  ← 点→资金Tab
│ └────────────────────────────┘      │
│ ┌────────────────────────────┐      │
│ │ 👥 成员   12人在营       → │      │  ← 点→成员Tab
│ └────────────────────────────┘      │
│                                      │
│ 📍 ●开营(7/20)──●展览(7/25)──●结营(7/27)│
└──────────────────────────────────────┘
```

### 跳转行为

| 卡片 | 点击后 | 实现方式 |
|------|--------|---------|
| 📋 我的任务 | 打开工作台，chip 默认选中 `[营队]` | `showMy({presetChip:'营队'})` |
| 🕐 今日 | 营地内切到日程 Tab，定位今天 | `switchCampTab('schedule')` |
| 💰 资金 | 营地内切到资金 Tab | `switchCampTab('funds')` |
| 👥 成员 | 营地内切到成员 Tab | `switchCampTab('members')` |

## 三、任务可见性

**不改结构，只改显示条件。** 工作台和村口任务大厅已有 chip 行 `[个人][营队][社区]`。

```
云村民/游客：  [全部范围] [个人] [社区]           ← 隐藏 [营队]
冒险者：      [全部范围] [个人] [营队] [社区]     ← 显示（只能看自己营）
共建者/管理员：[全部范围] [个人] [营队] [社区]     ← 全部显示
```

**信息分层：**

| 层级 | 内容 | 谁可见 |
|------|------|--------|
| 社区窗口 | 描述/照片/动态/送鼓励 | 所有人（外面的人看这里被吸引） |
| 任务大厅 | 个人委托 + 社区任务 | 所有人 |
| 任务大厅 | 营地任务 | 仅营地成员 |
| 营地内部 | 全部 6 Tab | 仅营地成员 |

```
│ └──────────────────────────────┘   │
│                                      │
│ [📅 完整时间线]                       │
```
