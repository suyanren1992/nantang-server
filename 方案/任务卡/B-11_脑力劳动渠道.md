# 任务卡 B-11：增加脑力劳动渠道（含一页设计，先确认后施工）

> 设计：Kimi Work · 施工：Claude Code · ⚠️ 需砚仁先确认设计再动手

## 一页设计

**问题**：现在上报劳动偏体力（打扫/农活/搬运），写方案、做设计、翻译、算账这类脑力劳动没有上报渠道，且脑力成果"看不见"，校核缺依据。

**方案（最小改动版）**：
1. 上报劳动的「劳动类型」增加一组脑力类：✍️写作 · 🎨设计 · 🌐翻译 · 📊策划 · 🧮算账（存在现有类型列表里，不新造分类体系）
2. 选脑力类时，表单**必填**「成果凭证」一栏：粘贴链接（文档/图片/表格地址）或文字说明成果在哪
3. 校核卡片上，脑力劳动显示凭证链接，校核者点开看完再作证——校核流程不变
4. 定价：走现有 NT 标准，本卡不设特殊价（后续如需要再调价目表）

**不做的事**（防止膨胀）：不做文件上传、不做脑力劳动专属校核规则、不改定价体系。

## 施工要点（确认设计后执行）
- 上报表单（ui-cardroom.js）：类型下拉加 5 项；选中时显示"成果凭证"必填输入
- 校核卡片：脑力类显示凭证（链接可点）
- 归档/时间线照常走（依赖 B-9 完成后的链路）

## 验收
- 能上报一条"✍️写作"+凭证链接，另一账号校核时能看到并点开凭证
- commit message：`feat(B-11): 脑力劳动上报渠道（5类+凭证必填）`
