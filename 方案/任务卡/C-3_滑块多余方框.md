# 复修卡 C-3：滑块左侧多余一个弧角方框（疑似多图层）

> 施工方：Kimi Code · 先查后改 · 单独 commit，不 push

## 现象
村落页滑块区域**屏幕左方有一个带弧角的多余方框**，像多出来的一个图层。滑块上正常的卡片框都在，但这一个是多余的。

## 排查方向
1. DOM 里 `#villageCarousel` 前后是否多了一个 `.vp-card` 或装饰性 div（A-5 改滚动定位后，原本藏在屏外的元素可能露出来）
2. CSS：`village-bg` / `village-bg-layer-2` / 伪元素 是否在该位置投了一个框
3. 是否 A-5 后第一张卡被 offset 挤出原位形成的"半个框"

## 修法
- 先确认这个框是什么元素（浏览器 DevTools 点选），结论写 BUG_TRACKER.md
- 多余 DOM → 移除（只移不删纪律：若是历史遗留元素，注释化或移档说明）；样式问题 → 改样式
- 不能影响三张正常卡片和滑动手感

## 验收
滑块区域无多余方框，三卡正常。
commit：`fix(C-3): 移除滑块左侧多余图层（真因:___）`
