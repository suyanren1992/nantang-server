## 铁律（三方都遵守）
1. 只移不删；一次一个目标；心脏文件（nt-core.js / app-data.js / api.js）的卡只由 Kimi Work 写
2. Claude Code 只 commit 不 push；push 由砚仁或 Kimi Code 执行（push = 上线 Render）
3. 同一时刻只有一方改代码，commit 后另一方才动手
