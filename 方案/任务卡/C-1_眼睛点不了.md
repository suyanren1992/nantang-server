# 复修卡 C-1：密码眼睛点不了（A-6 复修）

> 施工方：Claude Code · 单独 commit，不 push

## 现象
密码框旁出现了眼睛图标，但**点了没反应**，无法切换明文。用户要的就是"点眼睛看密码、确认没输错"。

## 排查方向（A-6 提交 723b36b 之后仍失效，重点怀疑）
1. 眼睛 span 的 onclick 是否被上层元素挡住（z-index/被 input 覆盖）
2. `togglePwdEye` 是否真的挂到了全局（script 加载顺序、函数名拼写）
3. 眼睛在 `position:absolute` 容器里是否随 input 样式塌陷成 0 宽

## 修法
- 浏览器实测点击必须生效：点一次 input type 变 text 能看明文，再点变回 password
- 图标给出按下反馈（👁/🙈 切换）
- 别只静态检查，自测通过再 commit

## 验收
注册页、登录页两处眼睛都可点、可看明文、可复原。
