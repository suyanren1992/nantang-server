# 复修卡 C-2：滑块居中后跳转太慢/有延迟（A-5 复修）

> 施工方：Claude Code · 单独 commit，不 push

## 现象
村落页滑动模块：居中是对了，但**滑动后归位/跳转速度明显变慢、有延迟**，以前没这么慢。

## 嫌疑（A-5 提交 997334d 引入）
1. scroll-snap-align:center 与新 scrollLeft 公式叠加，浏览器吸附动画被拉长
2. 圆点跳转 `behavior:'smooth'` 距离变长后显得肉
3. scroll 事件里每帧重算/改样式造成掉帧

## 修法方向（任选，以实测为准）
- 给 carousel 容器 CSS 加 `scroll-snap-type:x mandatory` 配合较短过渡；或跳转改 `behavior:'auto'` 自写 200ms 缓动
- scroll 监听器里的 DOM 查询移出循环、用缓存变量
- 目标：松手后 300ms 内归位，点圆点跳转干脆利落

## 验收
手机宽度下松手归位快、无拖泥带水；圆点跳转无可见延迟；居中保持正确。
