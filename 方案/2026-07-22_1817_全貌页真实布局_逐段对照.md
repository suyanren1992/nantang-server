## 每段展开后的子内容

### ⑤展开：在地人员 (_renderPresenceSection, line 461)

展示当前在线/在地的人员列表，含翻牌状态。

### ⑤展开：活跃营地 (_renderCampSection, line 479)

遍历 `AppData._data.camps`，过滤 `status==='active'`，每个营地一行：
```
🏕️ 南塘艺术共创营 👥8/16人 · 7/15-7/20  进入 ▸
```

### ⑤展开：时间线 (_renderTimelineSection, line 490)

合并 journal + announcements + verified verifications，去重排序，取最近20条：
```
7/21 15:30 📝 砚仁 打扫了画室
7/21 14:00 ✅ 大飞 收割番茄 · 砚仁校核 +15NT
```

## 真实的结构性问题

1. **poolCard 是空函数** — 第6段永远不渲染，社区池透明面板从未存在
2. **ic-card 不是方形** — 是普通圆角卡片（border-radius:8px），跟 mgmt-card-grid 的卡片不一样
3. **快捷入口只有3个** — 不是4个。厨房和冰箱合并为"放取物品"
4. **大扫除卡片点开是全屏弹窗** — `_showCardPopup(..., true)` 全屏模式
5. **田地卡片点开用的是 _showFieldSheet** — 不是 `renderFieldPanel`
6. **厨房冰箱卡片点开用的是 _showFridgeSheet** — 纯文字列表，没有任何冰箱门设计
7. **新手指引进度卡（_renderNewbieCard）没有被挂到任何section** — 定义了但没渲染
