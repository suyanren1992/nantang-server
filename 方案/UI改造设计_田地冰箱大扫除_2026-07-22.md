## 改造三：大扫除 — 补全可打扫空间

### 当前缺陷

`_collectCleaningRooms()` 只收集有 `r.cleaning && r.cleaning.length > 0` 标记的房间。

```javascript
// app.js:962-964 — 当前过滤逻辑
if (r.cleaning && r.cleaning.length > 0) {
  rooms.push(...);
}
```

这意味着：正厅、走廊、楼梯、洗手台、麻将室、阁楼等**没有 cleaning 标记的公共空间**——全都不会出现在打扫列表中。

### 改法

移除 cleaning 标记的前置条件，收集**所有建筑的除宿舍外的所有房间**：

```javascript
// 改后：所有非住宿房间都参与打扫
fKeys.forEach(function(fk) {
  (b.floors[fk] || []).forEach(function(r) {
    // 排除床位（住宿房间）
    if (r.id && r.id.indexOf('dorm') === 0) return;
    rooms.push({ id:r.id, name:r.name, icon:r.icon, status:st, buildingName:b.name });
  });
});
```

## 设计系统统一规则

所有 mgmt-sheet 面板统一使用以下结构顺序：

```
① mgmt-reminders   提醒横幅（警告/信息）
② mgmt-actions     操作按钮（主操作 + 辅助操作）
③ mgmt-quick-form  快速表单（toggle 展开/折叠）
④ mgmt-card-grid   卡片网格（2列，可点选）
⑤ mgmt-pricing     定价/说明（横排 chip）
⑥ mgmt-history     历史记录
```

**CSS class 清单**（已存在于 theme.css）：
- `.mgmt-reminders` / `.mr-item` (danger/warn/info/ok)
- `.mgmt-actions` / `.ma-btn` (primary/secondary)
- `.mgmt-quick-form` / `.qf-title` / `.qf-row` / `.qf-submit`
- `.mgmt-card-grid` / `.mgmt-card` / `.mc-icon` / `.mc-name` / `.mc-sub` / `.mc-nt`
- `.mgmt-pricing` / `.mgmt-price-chip`
- `.mgmt-history` / `.mh-item`
