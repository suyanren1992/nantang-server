## 一、全貌页(info) 布局层次

```
┌──────────────────────────────────────────────┐
│ 🎌 公告跑马灯 (announceTicker)              │  ← Layer 0
│ "砚仁打扫了画室 · 大飞收割了番茄 · ..."     │
└──────────────────────────────────────────────┘

┌──────────────────────────────────────────────┐
│ 🧹打扫  🛏️住宿  🌿田地  🍳厨房·冰箱       │  ← Layer 1: 快捷入口
└──────────────────────────────────────────────┘

┌──────────────┐  ┌──────────────────────────┐
│ 🃏 卡片室    │  │ ✓ 校核列表（待审核N条）  │  ← Layer 2: 校核+卡片室
│ 7天X张牌     │  │ 🧹 砚仁 打扫画室 +10NT  │
│ X张待揭      │  │ 📦 大飞 放入鸡蛋 +2NT   │
└──────────────┘  └──────────────────────────┘

┌────────┐ ┌────────┐ ┌────────┐ ┌──────────┐
│🧹大扫除│ │🛏️住宿 │ │🌿田地 │ │🍳厨房冰箱│  ← Layer 3: 四张管理卡片
│ 状态   │ │ 状态   │ │ 状态   │ │ 状态     │
└────────┘ └────────┘ └────────┘ └──────────┘
     ↓ 点击打开 mgmt-sheet（底部弹出面板）

┌──────────────────────────────────────────────┐
│ 👥 社区动态（手风琴折叠）                   │  ← Layer 4: 社区信息
│   ▸ 在地人员 (N人在线)                      │
│   ▸ 活跃营地 (N个)                          │
│   ▸ 时间线                                  │
└──────────────────────────────────────────────┘

┌──────────────────────────────────────────────┐
│ 💰 社区池子透明面板                         │  ← Layer 5: 经济面板
│ 总发行 · 流通 · 托管 · 营地 · 储备         │
└──────────────────────────────────────────────┘
```

**样式规律**：
- Layer 0-1: 最轻量——纯文字/emoji按钮，无圆角卡片
- Layer 2: 紧凑卡片——白底+细边框+小圆角(8-10px)+内嵌按钮
- Layer 3: 方形网格——2x2布局+统一高度+状态点+数值摘要
- Layer 4: 手风琴——可折叠section+折叠箭头+数字badge
- Layer 5: 数据面板——渐变背景+大字体数字+多列布局

## 二、管理卡片展开后的 mgmt-sheet 布局

每个管理卡片点击后弹出 `mgmt-sheet`（底部面板），内部结构统一：

```
┌─ mgmt-sheet ─────────────────────────────────┐
│                                              │
│  ⚠ 提醒区（warning/ok banner）              │
│  大扫除: "下次7/21 15:00 · 提前2h开放选位"  │
│  田地:   "A区番茄即将成熟 · B区C区空闲"      │
│  厨房:   "鸡蛋还剩1天 · 豆腐已过期"          │
│                                              │
│  📝 操作按钮                                │
│  [我打扫了] [选位置]                        │
│  [我做了农活]                               │
│  [放入物品] [取出/消耗]                     │
│                                              │
│  ┌─ 快速表单（toggle展开）────────────────┐  │
│  │ 选空间 + 选动作 + 备注 + [✓ 记录]     │  │
│  └────────────────────────────────────────┘  │
│                                              │
│  📊 数据卡片网格（mgmt-card-grid）           │
│  ┌──────┐ ┌──────┐ ┌──────┐                 │
│  │ 图标 │ │ 图标 │ │ 图标 │                 │
│  │ 名字 │ │ 名字 │ │ 名字 │                 │
│  │ 状态 │ │ 状态 │ │ 状态 │                 │
│  └──────┘ └──────┘ └──────┘                 │
│                                              │
│  📜 历史记录（mgmt-history）                 │
│  日期 · 人物 · 动作 · 详情                   │
│                                              │
└──────────────────────────────────────────────┘
```

**可复用模板**：提醒区+操作按钮+卡片网格+历史——这四个模块构成了**标准管理面板**，可直接套用到营地的住宿/订餐管理。

## 四、设计系统复用方案

### 4.1 营地住宿 → 套用全貌页住宿面板

全貌页 `_showStaySheet()` 已有完整的住宿UI：
- 房间卡片网格（2列，含图标/名字/空调/价格/入住人数）
- 床位选择器（点击展开，已占用显示为灰色✕，可选显示数字）
- 入住日历弹窗（ci-overlay样式）
- 退房按钮

**直接复用**到营地的 "🛏️ 住宿" 面板，改数据源（从营地预算读价格）。

### 4.2 营地订餐 → 参考 shared-kitchen.html 原型

`prototype/shared-kitchen.html` 的设计亮点：
- **冰箱门双面板**（上层/下层，带把手动画）
- **物品行**：图标 + 名字 + 元标签(新鲜/快过期/已过期) + 操作按钮(+/-)
- **烹饪模式切换**：底部出现 cook-summary 面板
- **区域卡片**：手风琴展开式 zone-card
- **底部操作栏**：做饭 / 放入物品 两个大按钮

**适用场景**：营地的 "🥬 订餐" 面板——菜单区（zone-card手风琴）+ 已选餐品（cook-summary）+ 提交按钮。

### 4.3 营地田间管理 → 参考全貌页 cleaning 面板

`renderCleaningPanel()` 的布局模式：
- 脏污程度卡片（绿色🟢/黄色🟡/红色🔴）
- 每张卡片含：空间名、脏污百分比、状态标签
- 打扫按钮 + NT奖励显示
- 历史记录

**田间管理改造**：
- 点击地块卡片 → 弹出该地块的详细面板（类似宿管房间展开）
- 可修改种植内容（作物类型、种植日期、预计收获日期）
- 每块地可以分成多个子区域（如A区-1、A区-2）
- 子区域各有独立的作物/状态/进度条

### 4.4 角色差异 UI

| 区域 | 管理员 | 共建者 | 冒险者 | 访客 |
|------|--------|--------|--------|------|
| 营地窗口 | 完整管理面板 + 编辑 | 任务管理 + 预算查看 | 报名/取消 + 查看任务 | 只能看公开信息 |
| 成员列表 | 可编辑角色 | 可查看全部 | 可查看全部 | 只能看自己 |
| 订餐/住宿 | 配置价格/菜单 | 查看+预定 | 查看+预定 | 不可访问 |
| 财务管理 | 预算/结算/流水 | 查看预算 | 查看自己的 | 不可访问 |

## 五、我的建议

### 5.1 田间管理改造

当前的农场只有一个 "我做了农活" 表单——太简略。建议：

**两步交互**：
1. 点击地块卡片 → 弹出该地块的详情面板（底部 sheet）
2. 面板内含：
   - 地块总览（面积、土壤状态、当前作物）
   - 子区域列表（如A区-1番茄、A区-2空闲）
   - 每子区域：修改作物、种植日期、状态
   - 操作按钮：种植/浇水/施肥/收割
   - 该地块的农事历史

**样式参考**：全貌页住宿面板的"选房间→展开床位→点床位→出日历"的渐进式展开模式。

**数据模型建议**：
```javascript
plots: [
  { id:'A', name:'A区', subplots: [
    { id:'A1', crop:'番茄', planted:'7/10', harvest:'8/10', status:'growing' },
    { id:'A2', crop:'', status:'idle' }
  ]}
]
```

### 5.2 冰箱→厨房面板改造

当前厨房面板直接是物品列表——建议改为 shared-kitchen.html 的**冰箱门+区域手风琴**布局：
1. 顶部：冰箱双门面板（上层蔬菜/下层肉类）
2. 中部：其他区域手风琴（干货架、调料柜、冰柜）
3. 底部：操作按钮

### 5.3 风格统一规则

| 组件 | 样式规范 |
|------|---------|
| 信息卡片 | `border-radius:10px; border:1px solid #d0d9ce; background:#fff; padding:10-14px` |
| 按钮(主) | `background:var(--green-primary); color:#fff; border-radius:8px; font-size:.68-.78rem` |
| 按钮(次) | `background:transparent; border:1px solid #d0d9ce; color:#5a6e5c` |
| 状态标签 | 绿底(#e8f5e8)正常 / 黄底(#fef8e8)注意 / 红底(#fde8e8)危险 |
| mgmt-sheet | 底部弹出，圆角16px 上，背景白，max-height:65vh |
| 手风琴 | ▸/▾ 箭头 + 折叠body + border-top分隔 |
| 数据网格 | 2列 grid，gap:6px |
