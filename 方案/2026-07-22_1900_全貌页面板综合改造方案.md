```javascript
if (r.cleaning && r.cleaning.length > 0)  // 所有27个房间的 cleaning 都是 []！
```

##五、田地管理面板改造

### 5.1 问题根因

田地管理有两套**独立的并行实现**：

| | 路径A：_showFieldSheet | 路径B：renderFieldPanel |
|---|---|---|
| 谁调用 | ic-card 点击 | openMgmt('field') |
| 内容 | 纯内联样式简单列表 | mgmt-* 标准CSS类 |
| 交互 | 无选择、无状态灯 | 无选择、无状态灯（同样缺失） |
| 用户看到的 | **这个** | 管理浮层 |

用户从 ic-card 点击看到的是**路径A（简化版）**，而大扫除的路径A和路径B走的是**同一个渲染函数**。

## 八、设计规范统一

所有管理面板（mgmt-sheet）统一使用 6 层结构：

```
① mgmt-reminders   提醒横幅（danger/warn/info/ok）
② mgmt-actions     操作按钮（主+辅+已选计数）
③ mgmt-quick-form  快速表单（toggle折叠）
④ mgmt-card-grid   卡片网格（2列，状态灯+图标+名字+副标题+NT+选中）
⑤ mgmt-pricing     定价 chips（横排）
⑥ mgmt-history     历史记录
```

CSS class 已全部存在于 `theme.css`，无需新增。
