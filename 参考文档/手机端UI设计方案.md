## 一、交互模型

### 导航层级
```
App Shell
  ├── Top Bar (48px, fixed)    ← 角色信息入口
  ├── Main Content (flex:1)     ← 当前 Tab 内容
  └── Bottom Nav (56px, fixed)  ← 4 Tab 切换
```

### 手势规范
| 手势 | 区域 | 行为 | 动画 |
|------|------|------|------|
| 点击 Tab 图标 | 底部导航 | 切换 Tab | crossfade 200ms |
| 上下滑动 | 内容区 | 滚动列表 | 自然 |
| 左右滑动 | 村口卡片 | 切换卡片 | snap 吸附 |
| 左右滑动 | 标签 chip 栏 | 横向滚动标签 | 自然 |
| 点击卡片 | 任务/物品/成员 | Push 到子页面 | slideInRight 250ms |
| ← 返回按钮 | 子页面左上角 | Pop 回上级 | slideOutRight 200ms |
| Android 返回键 | 全局 | 同 ← 返回 | — |
| 点击按钮 | 各处 | 打开底部浮层 | translateY 350ms cubic-bezier(.32,.72,0,1) |
| 下拉把手 | 浮层顶部 | 关闭浮层 | translateY 250ms ease-in |
| 点击蒙层 | 浮层背景 | 关闭浮层 | 同上 |
| 长按 | 茶馆帖子(自己) | 弹出删除选项 | — |

### Tab 间不互通滑动
- Tab 内容切换仅通过点击底部图标
- 避免和内容区上下滚动冲突

### 页面层级
```
Tab 根页面
  ├─ [Push →] 全屏子页面（有 ← 返回）
  │    └─ [Push →] 二级子页面（如：成员档案→成员详情）
  └─ [Sheet ↑] 底部浮层（不 Push，覆盖在当前页上）
```
