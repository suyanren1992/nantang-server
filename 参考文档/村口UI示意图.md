## 和主界面的关系

```
村口（独立全屏）                     主工作界面（Tab 栏）
─────────────                       ─────────────
🎨 水彩背景                          ⬜ 卡片白底
🎪 热闹·游戏感                       📋 工作·效率感
🤖 小云接待员                        （无NPC）
📋 看任务·发布·领（不能管理）         📋 看+领+管理+审核+结算
身份升级入口                         无升级入口（已有身份）
→ 点活动卡片进入主界面               → 齿轮菜单「回村口」
```

**村口的任务大厅**和**主界面的任务大厅 tab**是同一个 `renderQuestBoard()` 函数，但给不同的 `context`：
- `context='village'` → 橱窗模式：有小云、背景暗化、不显示 tab 栏
- `context='camp'` → 工作模式：当前的样子
