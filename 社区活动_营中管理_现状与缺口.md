## 移动端整合建议

43 个缺失元素整合为 5 个移动端页面。按角色区分：

### 管理员专属（3 个 overlay）

| 页面 | 整合哪些 PC 端 Tab |
|---|---|
| **营地管理** overlay | ntadmin (已发布/待审核/应付未付/NT池) + budget (资金双卡汇总) |
| **时间线** overlay | timeline (进度条+日历+里程碑) + taskhall (每日任务+任务总览) |
| **通关结算** overlay | settlement (头像+详情卡) + leaderboard (排行榜) |

### 所有人可见（2 个 overlay）

| 页面 | 整合哪些 PC 端 Tab |
|---|---|
| **成员** overlay | members (角色筛选+头像墙+详情) |
| **我的任务** Tab | tab-map (冒险者每日视图) → 已有的 `myTabTasks` |

---

## 优先级

```
P0 — 营地管理 overlay（管理员最核心的面板）
P1 — 时间线 overlay（进度跟踪）
P2 — 通关结算 overlay + 排行榜
P3 — 成员详情完善
```
