### 逻辑

```javascript
function renderCommunityHub() {
  // 从 data.camp_info 或 MOCK 数据读取活动列表
  // 进行中的活动 → 显示「进入营地」+「查看窗口」
  // 即将开始的 → 只显示「查看窗口」
  // 往期 → 显示「查看回顾」
  // 底部「世界终端」→ 仅 currentUser.role === 'admin' 时显示
```

### 按钮行为（简化后）

**原则：成员直接进，非成员看窗口。**

| 用户身份 | 点活动卡片 | 点「查看窗口」 |
|---|---|---|
| admin/builder/adventurer | → 直接进营地主页 | → 打开营地对外展示页 |
| npc/visitor/未登录 | → 打开营地对外展示页（底部有「我要报名」） | — |

**社区主页每个活动卡片只显示一个主按钮 + 一个小字链接：**

```
进行中的活动（成员）：
  [进入营地]            ← 主按钮，绿色

进行中的活动（非成员）：
  [查看窗口]  [我要报名 →] ← 主按钮 + 报名入口

即将开始的活动：
  [查看窗口]             ← 只有一个按钮

往期：
  [查看回顾]             ← 一个按钮
```

**管理员始终可以看到「进入营地」+ 底部「🌍 世界终端」。**

### 「我要报名」逻辑

```javascript
function applyForCamp(campId) {
  if (!currentUser) { /* 跳登录 */ return; }
  if (currentUser.role === 'visitor') {
    // 弹出邀请码输入框
    showInviteCodePrompt(campId);
    return;
  }
  if (currentUser.role === 'adventurer' || currentUser.role === 'builder' || currentUser.role === 'admin') {
    // 直接报名
    addToCamp(campId);
  }
}
```
