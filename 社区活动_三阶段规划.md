### 4.1 结营仪式（GSAP 动画）

管理员点「启动结营」→ 动画过渡 → 数据打包。

```
🔥 敬字亭火焰燃起
  → 「以字入火，以志成林」浮现
    → 火焰渐熄，化为星点
      → 数据报告自动生成
        → 结营报告页展示
```

**时长：** 约 5 秒。敬字亭是实景仪式——现实中大家在敬字亭烧掉练习稿和笔记。App 里的按钮是仪式的数字触发器。

## 七、先做什么后做什么

**先不做这个规划。** 当前执行窗口继续跑 05-10 的 UI 补齐。

**05-10 做完 → 数据层接入 → 再启动模块 A。**

理由：创营表单需要 `data` 对象（创建营队=写 `data.camp_info`），开营仪式需要 `data.tasks` 和 `data.members`（显示每人任务数），结营报告需要 `data.finance`（统计 NT/RMB）。没有数据层，仪式动画全是假数据——刷一下就没了。

**而数据层接入需要 UI 先做完。** 不然接入数据的时候发现某个页面布局崩了，不知道是数据问题还是 UI 问题。
