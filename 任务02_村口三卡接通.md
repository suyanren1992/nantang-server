# 任务 02 · 村口三卡接通

> **目标：** 村口三张轮播卡的「进入」按钮全部有反应。
> **改什么：** `nantang-mobile.html` 中村口卡片的 `onclick` 属性
> **不改：** CSS、JS 函数签名

---

## 当前状况

三张卡片的按钮都没有 onclick：

```html
<!-- 左卡 -->
<button class="vp-btn">📋 进入副本</button>           ← 点了没反应

<!-- 中卡 -->
<button class="vp-btn pri">🏛️ 去接任务</button>      ← 点了没反应

<!-- 右卡 -->
<button class="vp-btn">🗺️ 进入地图</button>           ← 点了没反应
```

---

## 要改成什么

每张卡点了之后打开对应的页面（新页面后续任务会创建）。

---

## 改什么

只改三个 `<button>` 的 onclick，加 `onclick="..."` 属性。

### 左卡「社区副本」— 第 175 行

```html
<button class="vp-btn" onclick="openCommunityPage()">📋 进入副本</button>
```

### 中卡「任务大厅」— 第 176 行

```html
<button class="vp-btn pri" onclick="openQuestHallPage()">🏛️ 去接任务</button>
```

### 右卡「实景地图」— 第 177 行

```html
<button class="vp-btn" onclick="openMapPage()">🗺️ 进入地图</button>
```

---

### 新增三个跳转函数

在内联 `<script>` 中加三个简单函数。目前目标页面还没有，先用子页面（subPage）做占位，后续任务再替换为完整页面。

```javascript
function openCommunityPage() {
  // 任务 03 会创建完整页面，现在先用 subPage 占位
  document.getElementById('subTitle').textContent = '社区副本';
  document.getElementById('subBody').innerHTML = '<div style="text-align:center;padding:40px;color:#5a6e5c">🏕️<br><br>活动列表即将上线</div>';
  document.getElementById('subPage').classList.add('open');
}

function openQuestHallPage() {
  // 任务 04 会创建完整页面
  document.getElementById('subTitle').textContent = '任务大厅';
  document.getElementById('subBody').innerHTML = '<div style="text-align:center;padding:40px;color:#5a6e5c">🏛️<br><br>任务大厅即将上线</div>';
  document.getElementById('subPage').classList.add('open');
}

function openMapPage() {
  // 任务 05 会嵌入完整地图
  document.getElementById('subTitle').textContent = '实景地图';
  document.getElementById('subBody').innerHTML = '<div style="text-align:center;padding:40px;color:#5a6e5c">🗺️<br><br>地图加载中…</div>';
  document.getElementById('subPage').classList.add('open');
}
```

---

## 验证

1. 打开页面 → 村口
2. 滑动到每张卡 → 点「进入」按钮
3. 右滑面板应弹出，显示占位文字
4. 点 `←` 返回，面板关闭
5. 三张卡都能打开、都能关闭
